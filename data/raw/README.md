## **Raw M5 Source Files**

The raw M5 source files are not included in this repository.

The three original Kaggle source files are not included in this repository due to usage restriction.
- `calendar.csv`
- `sales_train_validation.csv`
- `sell_prices.csv`

These files are pubicly available directly from the Kaggle competition page: "M5 Forecasting - Accuracy". 

Please download them directly from the **Kaggle competition page** (https://www.kaggle.com/competitions/m5-forecasting-accuracy/data).

Once downloaded, place the files in the **data/raw/** folder before running the first notebook: **01_data_collection_and_scope_selection.ipnyb**
