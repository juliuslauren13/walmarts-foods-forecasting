## **Processed Data Files**

The following files are included and allow the 03, 04, and 05 notebooks to run independently without re-running earlier steps:

| File | Description | Generated by | Input for Notebook |
|------|-------------|--------------|--------------------|
| `scope_selection.csv` | Top store (CA_3) and category (FOODS) | 01_data_collection_and_scope_selection.ipnyb | 02_data_preprocessing_and_eda_engineering.ipnyb |
| `store_dept_daily_sales.csv` | Department-level daily aggregation — CA_3 FOODS_1, FOODS_2, FOODS_3 | 02_data_preprocessing_and_eda_engineering.ipnyb |  03_feature_engineering.ipnyb |
| `modeling_dataset.csv` | Final modeling-ready dataset — 5,487 rows x 53 columns | 03_feature_engineering.ipnyb | 04_modeling_and_evaluation.ipynb |

**Note on `m5_merged_data.csv`**:

This file is 8.59 GB and cannot be hosted on GitHub. It is the intermediate output of  03_feature_engineering notebook— a merged and reformatted version of the three raw Kaggle source files.

To regenerate it:
1. Download the three raw files from Kaggle and place them in `data/raw/`
2. Run notebook 01 to 03 in sequence.
3. The file will be saved automatically to `data/processed/`
